[Unity](https://unity.com) costs $25/mo. If that's not in the budget yet, here are free options that get most of the job done.

## Quick comparison

| Tool | License | Platform | Best for |
|---|---|---|---|
| **[Godot Engine](https://godotengine.org)** | Open source | Mac, Windows, Linux | indie 2D/3D games, no royalty |
| **[Defold](https://defold.com)** | Open source | Mac, Windows, Linux | small 2D games shipped to mobile + web |

## Each alternative in detail

### Godot Engine

[Godot Engine](https://godotengine.org) is open source and serves the same workflow as Unity for most beginners use cases.

**Why it works as a Unity alternative**

- Open source
- No royalties ever
- Lightweight install

**Where it falls short of Unity**

If you need 3D commercial games, you'll eventually outgrow this.

### Defold

[Defold](https://defold.com) is open source and serves the same workflow as Unity for most beginners use cases.

**Why it works as a Unity alternative**

- Tiny binary builds
- No royalties
- Strong on mobile + HTML5

**Where it falls short of Unity**

If you need 3D commercial games, you'll eventually outgrow this.

## When you should just pay for Unity

If you're already making money from your work and Unity would save you 2+ hours/week, the math says pay for it. Free alternatives are right when you're validating an idea or learning the craft.
