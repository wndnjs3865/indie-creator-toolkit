You're picking between [Obsidian](https://obsidian.md) and [Logseq](https://logseq.com). Both are local-first. Both write to plain markdown files. Both have backlinks. The difference is editorial style — Obsidian is page-based (one document at a time), Logseq is outliner-based (bullet-tree where every line is a node). Here's how to pick.

## How to choose

- **Pick [Obsidian](https://obsidian.md)** if you draft, write, or output anything prose-shaped. Page-based editing is more natural for that work.
- **Pick [Logseq](https://logseq.com)** if you capture thoughts in bullets, you connect ideas via daily-journal entries, and you specifically want a Roam Research replacement.

## The math

Both are free. Cost is not the deciding factor here — both write the same plain markdown to your local disk, so you can experiment with one and migrate to the other without losing data.

The deciding factor is editorial style. If you draft essays in Obsidian, the cursor lands in a page. If you draft in Logseq, the cursor lands in a bullet — and that bullet wants to be short and connected to other bullets. The friction goes the wrong way for prose.

## Verdict

| Situation | Pick |
|---|---|
| Drafting essays / newsletter issues | **Obsidian** |
| Outliner-style note-taking | **Logseq** |
| Roam Research replacement | **Logseq** |
| Want largest plugin ecosystem | **Obsidian** |
| Care about open-source insurance | **Logseq** |
| Mobile-heavy workflow | **Obsidian** |
| Daily journal as your capture surface | **Logseq** |

If you're between the two — try Obsidian first if you write prose, Logseq first if you think in bullets. Your markdown files work in both, so the cost of switching is essentially zero.

## Honest disclosure

Both tools are free for personal use. Obsidian sells a paid sync add-on; Logseq is fully free + open source. Neither runs an affiliate program we're enrolled in. We compare them because they're the right pick for different readers, full stop.
