You're picking between [Godot Engine](https://godotengine.org) and [GameMaker](https://gamemaker.io). Both serve game engine workflows. Here's how they actually differ.

## Side-by-side

| | Godot Engine | GameMaker |
|---|---|---|
| Free tier | Fully free, MIT license | Free for non-commercial use |
| Starting price | Free | Free |
| Paid price | — | $80/mo |
| Platform | Mac, Windows, Linux | Mac, Windows |
| Open source | Yes | No |

## Where Godot Engine wins

- Open source
- No royalties ever
- Lightweight install

### Godot Engine downsides

- Smaller asset marketplace than Unity
- 3D less mature than 2D

## Where GameMaker wins

- Battle-tested for 2D
- Good drag-and-drop on-ramp

### GameMaker downsides

- Limited 3D capability
- Console export licenses are paid

## Verdict

- **Pick [Godot Engine](https://godotengine.org)** if you value: indie 2D/3D games, no royalty.
- **Pick [GameMaker](https://gamemaker.io)** if you value: 2D commercial games (Hyper Light Drifter, Undertale stack).

If you're starting from zero, the free tier of either lets you ship your first project before you commit. That's the right way to choose.
