## Where Audacity wins

- Free forever
- Plugin ecosystem
- Works offline

### Audacity downsides

- UI feels old
- Multitrack workflow clunky

## Where REAPER wins

- One-time license
- Powerful scripting + customization

### REAPER downsides

- Steep learning curve
- Default UI intimidating

## Verdict

- **Pick [Audacity](https://www.audacityteam.org)** if you value: zero-budget audio editing.
- **Pick [REAPER](https://www.reaper.fm)** if you value: pro-quality DAW on a tight budget.

If you're starting from zero, the free tier of either lets you ship your first project before you commit. That's the right way to choose.
