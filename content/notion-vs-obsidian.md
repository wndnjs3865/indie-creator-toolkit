## How to choose

- **Pick [Obsidian](https://obsidian.md)** if you write alone, you want plain markdown drafts you own forever, and the workflow is research/notes/drafts (not databases or team tasks).
- **Pick [Notion](https://notion.so)** if you collaborate, you embed lots of media, or you specifically rely on database views. Cloud-hosting is a feature when you have collaborators.

## The math

Both are free for solo personal use. Cost only diverges if you (a) collaborate (Notion adds per-user) or (b) want sync across devices on Obsidian (paid add-on $4/mo).

The deeper trade-off: file format ownership. Obsidian's markdown drafts work in any text editor, paste into Substack/Buttondown/Ghost without reformatting, survive any company shutdown. Notion's blocks are excellent inside Notion but feel gluey on export — that's the platform-vs-files trade.

## Verdict

| Situation | Pick |
|---|---|
| Solo writer, prose output | **Obsidian** |
| Collaborating with anyone | **Notion** |
| Need database views | **Notion** |
| Care about plain-file ownership | **Obsidian** |
| Embedding lots of media / mockups | **Notion** |
| Working offline often | **Obsidian** |
| Building a team wiki | **Notion** |
| Research with backlinks | Either — both have backlinks |

If you're between the two — try Obsidian first if you're solo and write prose. The free tier is unlimited and the markdown files transfer to anything later. Notion's switch later (if you start collaborating) is straightforward; the reverse migration (Notion → Obsidian) is harder once you've built Notion-specific databases.
