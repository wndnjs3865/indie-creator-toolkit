## How to choose in under 60 seconds

- **You ship code and want fast issue triage** → [Linear](https://linear.app). Keyboard-first, GitHub-integrated.
- **You want pure kanban, zero ceremony** → [Trello](https://trello.com).
- **You want tickets + GDD + wiki in one place** → [Notion](https://notion.so).

## Why I lead with Linear in 2026

Three reasons.

1. **Speed compounds.** Filing a bug in Linear takes 1-2 seconds with keyboard shortcuts. Across a year of indie dev, that speed difference equals hours back in your project.

2. **The free tier covers most indie work.** 10 users + 250 active issues is enough for a solo dev or a 5-person team for years. You don't graduate to paid until you have actual scale problems.

3. **GitHub integration closes the loop.** Mention `Closes ENG-42` in a PR, the issue auto-closes when merged. No manual ticket admin between commits.

If you've already tried Linear and hit its ceiling — drop me a note about what you needed instead. I update this list every quarter based on real reader feedback.

## What I deliberately left out

- **Jira** — industry standard, wrong scale. Solo dev workflows drown in Jira's configuration surface.
- **GitHub Issues / Projects** — fine if you're already deep in GitHub and want zero second-tool overhead. The tracker UX trails Linear noticeably.
- **ClickUp** — more flexible than Linear, but the flexibility becomes its own configuration tax.
- **Hacknplan** — game-dev-specific, but the user base is small and the UX feels stuck in 2018.
