You're picking between [Krita](https://krita.org) and [Procreate](https://procreate.com). They're both painting-first tools (not photo editors pretending to paint), but they live on different hardware. Krita is the free, open-source desktop tool that runs on Mac/Windows/Linux/Android. Procreate is the $13 iPad app with the best Apple Pencil integration in the business. Here's how to pick.

## How to choose

- **Pick [Krita](https://krita.org)** if you paint on desktop with a Wacom or XP-Pen tablet, you want free + open source, or you specifically need Linux support.
- **Pick [Procreate](https://procreate.com)** if iPad with an Apple Pencil is your primary painting setup. The Pencil integration and touch gestures are not replicated elsewhere — including Krita on iPad.

## The math

Krita is $0, forever. Procreate is $13 once. Across a 10-year career, both are essentially free.

The hardware question dominates. If you don't have an iPad + Apple Pencil, Procreate is moot. If you do have iPad + Pencil and you draw on it daily, Procreate's touch integration is genuinely worth $13.

## Verdict

| Situation | Pick |
|---|---|
| Desktop painter (Mac/Win/Linux) | **Krita** |
| iPad-first illustrator with Pencil | **Procreate** |
| Linux user | **Krita** (Procreate not available) |
| Need animation timeline (free) | **Krita** |
| Want best Pencil integration | **Procreate** |
| Care about open-source | **Krita** |
| Mix desktop + iPad workflow | Both — combined cost is $13 |

If you're between the two — they're complementary if you have iPad + desktop. Krita on desktop + Procreate on iPad is a strong combo for $13 total. If you only have one device, pick the tool that matches it.
